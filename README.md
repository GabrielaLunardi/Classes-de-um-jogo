## 🎮 Classes de um Jogo 

 O objetivo do projeto é criar uma classe genérica que representa um herói de uma aventura, com propriedades como nome, idade e tipo, além de um método de ataque que exibe mensagens diferentes conforme o tipo do herói.

---

## 📋 Funcionalidades

Criação de heróis com nome, idade e tipo com exibição de mensagem personalizada para cada tipo e tratamento para tipos desconhecidos.

- Tipos de heróis disponíveis:
  
  🗡️ `guerreiro` → ataca com **espada**  
  🧙 `mago` → ataca com **magia**  
  🥋 `monge` → ataca com **artes marciais**  
  🥷 `ninja` → ataca com **shuriken**
